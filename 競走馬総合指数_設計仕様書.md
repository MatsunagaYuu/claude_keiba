# 🏇 競走馬総合指数 設計仕様書

## ■ 目的

競走馬の「絶対的な強さ」を単一の総合指数として算出する。\
本指数は着順ではなく、フィジカル能力（スピード・キレ・スタミナ）を可視化することを目的とする。

------------------------------------------------------------------------

# ■ 強さの定義

強い馬とは：

> レース条件（展開・馬場・斤量・コース形状）が変化しても\
> 難所区間で高いパフォーマンスを再現できる馬。

強さ ＝ フィジカル能力の総合値

------------------------------------------------------------------------

# ■ 能力構成要素

## 1. スピード（SPD）

**定義：**\
高速巡航能力・追走能力

**評価区間：**\
前半3F + 中盤2F

**算出式：**

SPD =\
(基準前半3F − 実前半3F) × A\
+ (基準中盤2F − 実中盤2F) × B

------------------------------------------------------------------------

## 2. キレ（ACC）

**定義：**\
瞬発力・終盤の加速能力

**評価区間：**\
後半3F（芝）\
後半2F重視（坂コース）

**算出式：**

ACC =\
(基準後半3F − 実後半3F) × C

------------------------------------------------------------------------

## 3. スタミナ（STN）

**定義：**\
巡航速度を維持する能力・減速耐性

**評価方法：**\
中盤 → 終盤の減速率

**算出式：**

STN =\
(中盤平均 − 終盤平均) × D

減速が小さいほど高評価。

------------------------------------------------------------------------

# ■ 斤量補正

斤量補正 = (基準斤量 − 実斤量) × K

-   軽斤量での好走は減点\
-   重斤量での好走は加点

------------------------------------------------------------------------

# ■ 馬場補正

馬場状態（クッション値・含水率）を係数化し、指数に反映。

例：

-   良 = 1.00\
-   稍重 = 1.05\
-   重 = 1.10

------------------------------------------------------------------------

# ■ コース特性補正

コースごとに評価区間の重みを変更する。

### 芝

総合 = 0.4SPD + 0.4ACC + 0.2STN

### ダート

総合 = 0.6SPD + 0.2ACC + 0.2STN

------------------------------------------------------------------------

# ■ 絶対基準（例：芝1600m）

前半3F = 35.1\
中盤2F = 23.9\
後半3F = 35.3\
基準走破 = 1:34.3

------------------------------------------------------------------------

# ■ 最終総合指数

総合指数 =\
(SPD + ACC + STN)\
+ 斤量補正

最終的に馬場係数を掛け、偏差値化または1000基準化する。

------------------------------------------------------------------------

# ■ 指数の思想

-   着順は評価しない\
-   展開依存を排除する\
-   能力の再現性を評価する\
-   強い負けを拾う
-   芝/ダートで指数
-   距離ロスや不利など映像からのみ読み取れる情報は考慮しない\

------------------------------------------------------------------------
